# Implementação Front-end RestauranteUniversipratico

## Atividade

A atividade proposta consiste na implementação de algumas telas que abrangem os requisitos, que estão abordados em <a href="./Requisitos.md"> Requisitos.md</a>.

## Tecnologias

As tecnologias utilizadas serão React Native com Expo.

Para executar a tela deve-se:

(OBS: As dependências devem estar instaladas)

1. Entrar no diretório da tela

2. executar o comando

```shell
npx expo start
```

3. Baixar o aplicativo "Expo go" no dispositivo móvel
4. Ler o Qr-Code que aparecer na tela
